# odin-recipes
Odin Recipes Project

Project Overview:

The idea behind doing this project is to inculcate and experiment with our learnings of the basics of HTML.
It includes all the concepts taught under HTML in the curriculum of The Odin Project. 
The project hosts on its homepage and image a list of 3 recipes. The 3 recipes are linked to the Homepage.
Each Recipe page contains an image of the dish, multiple headings, the ingredients required to make the dish and the steps to follow.
Lastly, the recipe pages contain links to other recipes and a link back to the Homepage.

Key Learnings:

Usage of Git - HOW, WHEN & WHY to commit. Clearer understanding of basic git workflow.
Understanding & usage of various HTML Elements & Tags.
Usage of HTML Boilerplate.
Usage & implementation of Ordered & Unordered Lists.
Usage & implementation of Links & Images.
Usage of attributes concerning multiple elements & tags.
